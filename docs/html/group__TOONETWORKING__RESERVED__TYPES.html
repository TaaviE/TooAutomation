Detailed Description

Secret HMACs Reserved message header types

System types are 33-90 ('!'-A'-'Z'), they must not be used outside the library
User types that are acknowledged are 91-127 ('['-'a'-'z'-'DEL') and 1-32 ('^@'-' ') are not acknowledged
